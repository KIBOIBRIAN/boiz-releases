# Boiz - Releases

This repository contains the official release builds for **Boiz** (KianAI) - an AI-powered Interview and Meeting Assistant.

## Downloads

All releases are available in the [Releases](https://github.com/boiz-app/boiz-releases/releases) section.

### Supported Platforms

| Platform | File Types |
|----------|------------|
| **Windows** | `.exe` (Installer) |


## Installation

### Windows
1. Download `Boiz-Setup-{version}.exe` from the latest release
2. Run the installer
3. Follow the installation wizard
